reclab.utils.check_permitted_value¶

reclab.utils.check_permitted_value(permitted_dict, provided_key)[source][source]¶

Get a key/value mapping from a dict of permitted value.

If the key is not found in the dictionary, raise a KeyError with a more descriptive message of the error.

Parameters:

permitted_dict : dict

A dictionary of valid key/value mappings

provided_key : str, int

The key provided by the user
