API Reference¶

This is the class and function reference for reclab. Please refer to the full user guide for further details, as the class and function raw specifications may not be enough to give full guidelines on their uses.


reclab.base: Base classes and functions for reclab¶

Base class for recommenders and other functions used package-wide.


Base recommender class¶

base.BaseRecommender() Base class for all recommenders.

Common-use functions¶

base.clone(estimator[, safe, clone_model_key]) Constructs a new estimator with the same parameters.
base._recommend_items_and_maybe_scores(best, …) Internal method for recommender inference.

reclab.collab: Collaborative filtering in reclab¶

The reclab.collab sub-module defines a number of collaborative filtering recommender algorithms, including popular matrix factorization techniques and nearest neighbor methods.


Collaborative filtering estimators¶

collab.AlternatingLeastSquares([factors, …]) Alternating Least Squares
collab.AnnoyAlternatingLeastSquares([…]) Alternating Least Squares with nearest neighbor indexing.
collab.ItemItemRecommender([metric, k, k1, …]) Item-item collaborative filtering.
collab.NMSAlternatingLeastSquares([factors, …]) Alternating Least Squares with nearest neighbor indexing.

reclab.datasets: Benchmarking ratings data for recommenders¶

The reclab.datasets submodule provides several different ratings datasets used in various examples and tests across the package. If you would like to prototype a model, this is a good place to find easy-to-access data.


Dataset loading functions¶

datasets.load_lastfm([cache, as_sparse, dtype]) Load and return the lastFM dataset.

reclab.metrics: Ranking metrics for scoring recommenders¶

The reclab.metrics submodule provides several different rankings metrics that are widely used for benchmarking the efficacy of a recommender algorithm.


Ranking metrics¶

metrics.mean_average_precision(predictions, …) Compute the mean average precision on predictions and labels.
metrics.ndcg_at(predictions, labels[, k]) Compute the normalized discounted cumulative gain at K.
metrics.precision_at(predictions, labels[, k]) Compute the precision at K.

reclab.model_deployment: Model deployment tools for recommenders¶

The reclab.model_deployment submodule provides tools for serving your selected model and producing recommendations all within a wrapper, including:

  • Automatically encoding/inverse-encoding user and product IDs
  • Producing recommendations from JSON or array (depending on your service architecture)

Deploy wrapper¶

model_deployment.RecommenderDeployment(estimator) A wrapper for deployed recommender models.

reclab.model_selection: Model selection tools for recommenders¶

The reclab.model_selection submodule provides many utilities for cross- validating your recommender models, splitting your data into train/test splits and performing grid searches.


Data splitting¶

model_selection.train_test_split(X[, …]) Create a train/test split for sparse ratings matrices.

Cross validation¶

model_selection.KFold([n_splits, …]) K-fold cross validation.

Grid searches¶

model_selection.RandomizedRecommenderSearchCV(…) Randomized search on hyper parameters.
model_selection.RecommenderGridSearchCV(…) Exhaustive search on hyper parameters.

reclab.utils: Utilities¶

Utilities and validation functions used commonly across the package. As much as is possible, the functions within the utilities directory should be used across the package in a repeatable fashion.


Validation & array-checking functions¶

utils.check_consistent_length(u, i, r) Ensure users, items, and ratings are all of the same dimension.
utils.check_permitted_value(permitted_dict, …) Get a key/value mapping from a dict of permitted value.
utils.check_sparse_array(array[, dtype, …]) Input validation on a sparse matrix.
utils.get_n_factors(n_dim, n_factors) Get the number of factors to compute.
utils.inherit_function_doc(parent) Inherit a parent instance function’s documentation.
utils.is_iterable(x) Determine whether an element is iterable.
utils.safe_mkdirs(directory) Make an entire directory tree.
utils.to_sparse_csr(u, i, r[, axis, dtype]) Create a sparse ratings matrix.
